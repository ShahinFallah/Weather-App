# Local Installation

#### Download the projects using git

```
$ git clone https://github.com/ShahinFallah/react-projects.git
$ cd weather-app
```

## How to start coding?

- Install the packages with  `$ npm install`.
- Run the webpack server with  `$ npm run dev`
